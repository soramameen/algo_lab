# CLRS 2.3 Designing Algorithms — 翻訳・整理・Notion保存 計画

CLRS（Introduction to Algorithms）第2章 §2.3「Designing Algorithms」の本文を翻訳し、
勉強用に整理した上で Notion の `algo memo` データベースに保存する。

## 3. 方針

- **翻訳方針**: 英語原文を日本語に翻訳する。専門用語は业界標準の訳語を使用し、
  初見でも意味が通るように簡潔な注記を添える。
- **タイトル**: ページタイトルは `CLRS 2.3 Designing Algorithms` とする。
- **資料プロパティ**: Notion ページのプロパティに上記タグ・ステータスを設定する。

## 4. 実行フェーズ

### Phase 1: 翻訳

1. CLRS 2.3 の原文（英語）を取得する。
2. 段落・セクションごとに日本語に翻訳する。
3. 専門用語の訳語を統一する（merge, divide-and-conquer など）。

### Phase 2: 勉強用に整える

1. 翻訳テキストを見出し・箇条書き・コードブロック等で構造化する。
2. 疑似コード（MERGE-SORT 等）はコードブロックで記述し、
   各行にコメントで解説を付ける。
3. 重要な概念・定理には **太字** で強調し、理解しやすくする。

### Phase 3: Notion に保存

1. Notion の `algo memo` データベースを特定する。
2. タイトル `CLRS 2.3 Designing Algorithms` で新規ページを作成する。
3. Phase 2 で整えた内容をページ本文に書き込む。
4. プロパティ（タグ・ステータス）を設定する。
